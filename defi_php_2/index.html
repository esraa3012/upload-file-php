Script PHP

Réaliser le script php qui va permettre l'upload du fichier sur le serveur. Vous pouvez utiliser Ajax si vous voulez. Pas de librairie. C'est votre code php. En fonction de votre script vous aurez peut-être besoin de modifier des attributs et peut être ajouter des balises html pour afficher le message que le fichier à bien été uploader. Possibilité aussi d'ajouter des fichiers php.
